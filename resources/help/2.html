How to use TODO-APP
Simply open the App and create a new List(Group) or use the default list named "new list"
then start writing a new tasks in the bellow field and press Enter or Click Add button to add Your ToDo.
ToDos States
Each new created ToDo Has three states
  1. Pending state:Means Waiting...
  2. Done state:Means Finished...
  3. Doing state:Means on-going...

Toggling between Pending and Done states is through clicking the left mouse button
Setting state to Doing is through the middle mouse button.
ToDos can be Imported and Exported as JSON and TXT formats.
Other Features can be explored through the right click pop-up menu and Edit menu on the top.
